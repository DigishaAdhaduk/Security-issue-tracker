# ApniSec Security Issue Tracker

A full-stack security issue management platform built as part of the ApniSec SDE Intern assignment.

This application allows users to register and manage cybersecurity issues such as Cloud Security findings, VAPT reports, and Red Team assessments. The project emphasizes clean architecture, strict Object-Oriented backend design, and a simple professional UI.
